# SecShare Logo Kit

Variant: shield + protected link.

## Contents

- `svg/secshare-mark.svg` — source mark, transparent background.
- `svg/secshare-app-icon.svg` — rounded primary app/favicon icon.
- `svg/secshare-app-icon-light.svg` — light app icon variant close to the selected concept.
- `svg/secshare-logo-horizontal.svg` — horizontal logo lockup.
- `svg/secshare-logo-horizontal-mono.svg` — one-color horizontal logo.
- `png/mark/*` — transparent PNG mark exports, including 1024x1024.
- `png/app-icon/*` — PNG app/favicon exports, including 1024x1024.
- `png/logo/*` — horizontal logo PNG previews.
- `favicon/favicon.ico` — ICO containing 16x16, 32x32, 48x48.
- `favicon/favicon.svg` — SVG favicon.
- `favicon/apple-touch-icon.png` — 180x180.
- `favicon/android-chrome-192x192.png` — 192x192.
- `favicon/android-chrome-512x512.png` — 512x512.
- `favicon/android-chrome-1024x1024.png` — 1024x1024.
- `favicon/secshare-icon-120x120.png` — 120x120 PNG.
- `favicon/secshare-icon-1024x1024.png` — 1024x1024 PNG.
- `favicon/site.webmanifest` — minimal PWA/browser manifest.

## HTML snippet

```html
<!-- SecShare favicon / app icons -->
<link rel="icon" href="/favicon.ico" sizes="any">
<link rel="icon" href="/favicon.svg" type="image/svg+xml">
<link rel="apple-touch-icon" href="/apple-touch-icon.png">
<link rel="manifest" href="/site.webmanifest">
<meta name="theme-color" content="#2563EB">
```
